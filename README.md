# Outformasi Site

Static landing page untuk konsep:

**Gratis:** user belajar biasa.  
**Premium:** user menghasilkan artefak nyata.

## Struktur file

```text
outformasi-site/
├── index.html
├── styles.css
├── app.js
└── README.md
```

## Cara upload ke GitHub

1. Buka repository GitHub yang sudah dibuat.
2. Klik **Add file** → **Upload files**.
3. Upload semua file di folder ini: `index.html`, `styles.css`, `app.js`, `README.md`.
4. Klik **Commit changes**.

## Cara deploy ke Vercel

1. Buka Vercel.
2. Klik **Add New Project**.
3. Import repository GitHub.
4. Framework preset: **Other** atau biarkan otomatis.
5. Build command: kosongkan.
6. Output directory: kosongkan.
7. Klik **Deploy**.
